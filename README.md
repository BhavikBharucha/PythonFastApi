# PythonFastApi
Python FastAPI CRUD Application

A comprehensive FastAPI project demonstrating CRUD operations with secure authentication, authorization, and data validation. This project uses MySQL as the database and includes JWT authentication, refresh token implementation, role-based access control, user account lockout after multiple failed login attempts, file upload management, Pydantic schema validation, and custom input validation. It follows FastAPI best practices for building secure, scalable, and maintainable REST APIs.

Key Features:

CRUD Operations (Create, Read, Update, Delete)
MySQL Database Integration
JWT Authentication & Authorization
Refresh Token Implementation
Role-Based Access Control (RBAC)
User Account Lockout on Multiple Failed Login Attempts
File Upload and Management
Pydantic Request/Response Validation
Custom Input Validation
SQLAlchemy ORM Integration
RESTful API Development with FastAPI
